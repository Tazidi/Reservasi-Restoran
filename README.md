# **Cara Instalasi**
```
git clone https://github.com/Tazidi/Reservasi-Restoran.git

cp .env.example .env

composer install

npm install

php artisan migrate

php artisan db:seed --class=Admin

php artisan key:generate

npm run dev

php artisan serve
```
